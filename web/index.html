sph2wav

Flatten a NIST SPHERE file into a plain WAV — in your browser.

Please don't use this if a real tool can do the job. sph2pipe, ffmpeg, and sox all decode SPHERE. But we know they're old and awkward to install — so if that's in your way, drop a .sph file below and get a .wav back. Everything runs locally in this page; your file is never uploaded anywhere.
